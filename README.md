## 网义新闻发布系统，更新完成
新闻发布后台编辑系统采用[ueditor](http://ueditor.baidu.com/website/)
采用开发环境**Eclipse+Mysql+Tomcat7.0**
### 实现主要功能样式（详细见Image文件夹）
### 客户端功能  
新闻的浏览，包括图片新闻、今日头条、热点新闻
### 服务端功能   
- 新闻添加
- 新闻更新
- 类别添加
- 类别更新
- 评论更新
- 用户更新
